If things break

  • Engine not READY. .venv/bin/rai reasoners list to confirm. The gate auto-resumes them.
  • Stale notebook in Snowsight. .venv/bin/python prep_demo.py --skip-chat --skip-figures re-uploads it.
  • Agent answering with stale schema. .venv/bin/python -m agent.deploy update refreshes sprocs without re-registering.
  • Solver returning INFEASIBLE. Not a runtime problem - your formulation drifted. Check data/{{DOMAIN}}_demo_validation.sql first; if the data changed under you, regenerate. Otherwise the formulation needs /rai-prescriptive-problem-formulation.
  • Snowflake credentials. snow connection test -c rai. Profile lives in ~/.snowflake/connections.toml.
